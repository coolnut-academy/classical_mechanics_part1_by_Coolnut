บทพิสูจน์สมการคาบการสั่นของสปริงที่มีมวล

พิสูจน์โดยใช้หลักการอนุรักษ์พลังงานกล (Conservation of Mechanical Energy)

1

คำนวณพลังงานจลน์รวมของระบบ

(Total Kinetic Energy, $KE_{total}$)

พลังงานจลน์รวม คือผลบวกของพลังงานจลน์ของมวล $M$ และพลังงานจลน์ของสปริง $m$

พลังงานจลน์ของมวล M:

$$KE_M = \frac{1}{2}Mv^2$$

พลังงานจลน์ของสปริง (KE_m):

ตามโจทย์กำหนดให้ $x$ คือความยาวของสปริง และ $v$ คือความเร็วของมวล M ณ ขณะนั้น เราจะอินทิเกรตพลังงานจลน์ของชิ้นส่วนเล็กๆ ตลอดความยาวสปริง

  • ความเร็วของสปริงที่ตำแหน่ง $\xi$: $v_\xi = \frac{\xi}{x}v$
  • มวลของสปริงชิ้นเล็กๆ: $dm = \frac{m}{x}d\xi$

พลังงานจลน์ของสปริงทั้งเส้นคือ:

$$KE_m = \int_{0}^{x} \frac{1}{2}(dm)(v_\xi)^2 = \int_{0}^{x} \frac{1}{2}\left(\frac{m}{x}d\xi\right)\left(\frac{\xi}{x}v\right)^2$$

$$KE_m = \frac{1}{2}\frac{mv^2}{x^3} \int_{0}^{x} \xi^2 d\xi = \frac{1}{2}\frac{mv^2}{x^3} \left[\frac{\xi^3}{3}\right]_0^x = \frac{1}{2}\left(\frac{m}{3}\right)v^2$$

พลังงานจลน์รวมของระบบ:

$$KE_{total} = KE_M + KE_m = \frac{1}{2}Mv^2 + \frac{1}{2}\left(\frac{m}{3}\right)v^2$$

$$KE_{total} = \frac{1}{2}\left(M + \frac{m}{3}\right)v^2$$

2

คำนวณพลังงานศักย์ของระบบ

(Potential Energy, PE)

ในขั้นตอนนี้ เราจะใช้สัญลักษณ์ $x$ ตามความหมายมาตรฐานของ SHM คือ "ระยะกระจัดจากตำแหน่งสมดุล" พลังงานศักย์ของระบบที่เกี่ยวข้องกับการสั่นคือพลังงานศักย์ยืดหยุ่น

พลังงานศักย์ยืดหยุ่น:

$$PE = \frac{1}{2}kx^2$$

3

ใช้กฎการอนุรักษ์พลังงาน

พลังงานรวม $E$ ของระบบมีค่าคงที่ ดังนั้นอนุพันธ์ของพลังงานเทียบกับเวลาต้องเป็นศูนย์

$$E = KE_{total} + PE = \frac{1}{2}\left(M + \frac{m}{3}\right)v^2 + \frac{1}{2}kx^2 = \text{Constant}$$

$$\frac{dE}{dt} = \frac{d}{dt}\left[\frac{1}{2}\left(M + \frac{m}{3}\right)v^2 + \frac{1}{2}kx^2\right] = 0$$

4

หาสมการการเคลื่อนที่และคาบการสั่น (T)

หาอนุพันธ์ของสมการพลังงานเทียบกับเวลา $t$ โดยที่ $v = \frac{dx}{dt}$ และ $a = \frac{dv}{dt}$

$$\left(M + \frac{m}{3}\right)v\frac{dv}{dt} + kx\frac{dx}{dt} = 0$$

$$\left(M + \frac{m}{3}\right)va + kxv = 0$$

จัดรูปสมการการเคลื่อนที่:

$$a = -\frac{k}{\left(M + \frac{m}{3}\right)}x$$

เมื่อเปรียบเทียบกับสมการมาตรฐานของ SHM ($a = -\omega^2x$) จะได้:

$$\omega^2 = \frac{k}{M + \frac{m}{3}}$$

จากความสัมพันธ์ $T^2 = \frac{4\pi^2}{\omega^2}$ จะได้:

$$T^2 = \frac{4\pi^2}{\frac{k}{M + \frac{m}{3}}}$$

🎉 สมการคาบการสั่นสุดท้าย 🎉

$$ T^2 = \frac{4\pi^2}{k}\left(M + \frac{1}{3}m\right) $$
