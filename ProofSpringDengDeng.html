1

‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏£‡∏ß‡∏°‡∏Ç‡∏≠‡∏á‡∏£‡∏∞‡∏ö‡∏ö

(Total Kinetic Energy, $KE_{total}$)

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏£‡∏ß‡∏° ‡∏Ñ‡∏∑‡∏≠‡∏ú‡∏•‡∏ö‡∏ß‡∏Å‡∏Ç‡∏≠‡∏á‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏Ç‡∏≠‡∏á‡∏°‡∏ß‡∏• $M$ ‡πÅ‡∏•‡∏∞‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏Ç‡∏≠‡∏á‡∏™‡∏õ‡∏£‡∏¥‡∏á $m$

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏Ç‡∏≠‡∏á‡∏°‡∏ß‡∏• M:

$$KE_M = \frac{1}{2}Mv^2$$

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏Ç‡∏≠‡∏á‡∏™‡∏õ‡∏£‡∏¥‡∏á (KE_m):

‡∏ï‡∏≤‡∏°‡πÇ‡∏à‡∏ó‡∏¢‡πå‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡πÉ‡∏´‡πâ $x$ ‡∏Ñ‡∏∑‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡∏¢‡∏≤‡∏ß‡∏Ç‡∏≠‡∏á‡∏™‡∏õ‡∏£‡∏¥‡∏á ‡πÅ‡∏•‡∏∞ $v$ ‡∏Ñ‡∏∑‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡∏Ç‡∏≠‡∏á‡∏°‡∏ß‡∏• M ‡∏ì ‡∏Ç‡∏ì‡∏∞‡∏ô‡∏±‡πâ‡∏ô ‡πÄ‡∏£‡∏≤‡∏à‡∏∞‡∏≠‡∏¥‡∏ô‡∏ó‡∏¥‡πÄ‡∏Å‡∏£‡∏ï‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏Ç‡∏≠‡∏á‡∏ä‡∏¥‡πâ‡∏ô‡∏™‡πà‡∏ß‡∏ô‡πÄ‡∏•‡πá‡∏Å‡πÜ ‡∏ï‡∏•‡∏≠‡∏î‡∏Ñ‡∏ß‡∏≤‡∏°‡∏¢‡∏≤‡∏ß‡∏™‡∏õ‡∏£‡∏¥‡∏á

  • ‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡∏Ç‡∏≠‡∏á‡∏™‡∏õ‡∏£‡∏¥‡∏á‡∏ó‡∏µ‡πà‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á $\xi$: $v_\xi = \frac{\xi}{x}v$
  • ‡∏°‡∏ß‡∏•‡∏Ç‡∏≠‡∏á‡∏™‡∏õ‡∏£‡∏¥‡∏á‡∏ä‡∏¥‡πâ‡∏ô‡πÄ‡∏•‡πá‡∏Å‡πÜ: $dm = \frac{m}{x}d\xi$

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏Ç‡∏≠‡∏á‡∏™‡∏õ‡∏£‡∏¥‡∏á‡∏ó‡∏±‡πâ‡∏á‡πÄ‡∏™‡πâ‡∏ô‡∏Ñ‡∏∑‡∏≠:

$$KE_m = \int_{0}^{x} \frac{1}{2}(dm)(v_\xi)^2 = \int_{0}^{x} \frac{1}{2}\left(\frac{m}{x}d\xi\right)\left(\frac{\xi}{x}v\right)^2$$

$$KE_m = \frac{1}{2}\frac{mv^2}{x^3} \int_{0}^{x} \xi^2 d\xi = \frac{1}{2}\frac{mv^2}{x^3} \left[\frac{\xi^3}{3}\right]_0^x = \frac{1}{2}\left(\frac{m}{3}\right)v^2$$

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏à‡∏•‡∏ô‡πå‡∏£‡∏ß‡∏°‡∏Ç‡∏≠‡∏á‡∏£‡∏∞‡∏ö‡∏ö:

$$KE_{total} = KE_M + KE_m = \frac{1}{2}Mv^2 + \frac{1}{2}\left(\frac{m}{3}\right)v^2$$

$$KE_{total} = \frac{1}{2}\left(M + \frac{m}{3}\right)v^2$$

2

‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏®‡∏±‡∏Å‡∏¢‡πå‡∏Ç‡∏≠‡∏á‡∏£‡∏∞‡∏ö‡∏ö

(Potential Energy, PE)

‡πÉ‡∏ô‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ô‡∏µ‡πâ ‡πÄ‡∏£‡∏≤‡∏à‡∏∞‡πÉ‡∏ä‡πâ‡∏™‡∏±‡∏ç‡∏•‡∏±‡∏Å‡∏©‡∏ì‡πå $x$ ‡∏ï‡∏≤‡∏°‡∏Ñ‡∏ß‡∏≤‡∏°‡∏´‡∏°‡∏≤‡∏¢‡∏°‡∏≤‡∏ï‡∏£‡∏ê‡∏≤‡∏ô‡∏Ç‡∏≠‡∏á SHM ‡∏Ñ‡∏∑‡∏≠ "‡∏£‡∏∞‡∏¢‡∏∞‡∏Å‡∏£‡∏∞‡∏à‡∏±‡∏î‡∏à‡∏≤‡∏Å‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏™‡∏°‡∏î‡∏∏‡∏•" ‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏®‡∏±‡∏Å‡∏¢‡πå‡∏Ç‡∏≠‡∏á‡∏£‡∏∞‡∏ö‡∏ö‡∏ó‡∏µ‡πà‡πÄ‡∏Å‡∏µ‡πà‡∏¢‡∏ß‡∏Ç‡πâ‡∏≠‡∏á‡∏Å‡∏±‡∏ö‡∏Å‡∏≤‡∏£‡∏™‡∏±‡πà‡∏ô‡∏Ñ‡∏∑‡∏≠‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏®‡∏±‡∏Å‡∏¢‡πå‡∏¢‡∏∑‡∏î‡∏´‡∏¢‡∏∏‡πà‡∏ô

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏®‡∏±‡∏Å‡∏¢‡πå‡∏¢‡∏∑‡∏î‡∏´‡∏¢‡∏∏‡πà‡∏ô:

$$PE = \frac{1}{2}kx^2$$

3

‡πÉ‡∏ä‡πâ‡∏Å‡∏é‡∏Å‡∏≤‡∏£‡∏≠‡∏ô‡∏∏‡∏£‡∏±‡∏Å‡∏©‡πå‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô

‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡∏£‡∏ß‡∏° $E$ ‡∏Ç‡∏≠‡∏á‡∏£‡∏∞‡∏ö‡∏ö‡∏°‡∏µ‡∏Ñ‡πà‡∏≤‡∏Ñ‡∏á‡∏ó‡∏µ‡πà ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô‡∏≠‡∏ô‡∏∏‡∏û‡∏±‡∏ô‡∏ò‡πå‡∏Ç‡∏≠‡∏á‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏Å‡∏±‡∏ö‡πÄ‡∏ß‡∏•‡∏≤‡∏ï‡πâ‡∏≠‡∏á‡πÄ‡∏õ‡πá‡∏ô‡∏®‡∏π‡∏ô‡∏¢‡πå

$$E = KE_{total} + PE = \frac{1}{2}\left(M + \frac{m}{3}\right)v^2 + \frac{1}{2}kx^2 = \text{Constant}$$

$$\frac{dE}{dt} = \frac{d}{dt}\left[\frac{1}{2}\left(M + \frac{m}{3}\right)v^2 + \frac{1}{2}kx^2\right] = 0$$

4

‡∏´‡∏≤‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏Å‡∏≤‡∏£‡πÄ‡∏Ñ‡∏•‡∏∑‡πà‡∏≠‡∏ô‡∏ó‡∏µ‡πà‡πÅ‡∏•‡∏∞‡∏Ñ‡∏≤‡∏ö‡∏Å‡∏≤‡∏£‡∏™‡∏±‡πà‡∏ô (T)

‡∏´‡∏≤‡∏≠‡∏ô‡∏∏‡∏û‡∏±‡∏ô‡∏ò‡πå‡∏Ç‡∏≠‡∏á‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏û‡∏•‡∏±‡∏á‡∏á‡∏≤‡∏ô‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏Å‡∏±‡∏ö‡πÄ‡∏ß‡∏•‡∏≤ $t$ ‡πÇ‡∏î‡∏¢‡∏ó‡∏µ‡πà $v = \frac{dx}{dt}$ ‡πÅ‡∏•‡∏∞ $a = \frac{dv}{dt}$

$$\left(M + \frac{m}{3}\right)v\frac{dv}{dt} + kx\frac{dx}{dt} = 0$$

$$\left(M + \frac{m}{3}\right)va + kxv = 0$$

‡∏à‡∏±‡∏î‡∏£‡∏π‡∏õ‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏Å‡∏≤‡∏£‡πÄ‡∏Ñ‡∏•‡∏∑‡πà‡∏≠‡∏ô‡∏ó‡∏µ‡πà:

$$a = -\frac{k}{\left(M + \frac{m}{3}\right)}x$$

‡πÄ‡∏°‡∏∑‡πà‡∏≠‡πÄ‡∏õ‡∏£‡∏µ‡∏¢‡∏ö‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏Å‡∏±‡∏ö‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏°‡∏≤‡∏ï‡∏£‡∏ê‡∏≤‡∏ô‡∏Ç‡∏≠‡∏á SHM ($a = -\omega^2x$) ‡∏à‡∏∞‡πÑ‡∏î‡πâ:

$$\omega^2 = \frac{k}{M + \frac{m}{3}}$$

‡∏à‡∏≤‡∏Å‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå $T^2 = \frac{4\pi^2}{\omega^2}$ ‡∏à‡∏∞‡πÑ‡∏î‡πâ:

$$T^2 = \frac{4\pi^2}{\frac{k}{M + \frac{m}{3}}}$$

üéâ ‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏Ñ‡∏≤‡∏ö‡∏Å‡∏≤‡∏£‡∏™‡∏±‡πà‡∏ô‡∏™‡∏∏‡∏î‡∏ó‡πâ‡∏≤‡∏¢ üéâ

$$ T^2 = \frac{4\pi^2}{k}\left(M + \frac{1}{3}m\right) $$
