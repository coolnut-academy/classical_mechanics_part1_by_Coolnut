รวมเฉลยโจทย์กลศาสตร์

โดยวิทยากร: นายสาธิต ศิริวัชน์ (Coolnut) | คลิกที่โจทย์แต่ละข้อเพื่อดูเฉลยแบบละเอียด

ความคืบหน้า 0/10

โจทย์: รถยนต์คันหนึ่งเคลื่อนที่เป็นเส้นตรงจากหยุดนิ่ง โดยเพิ่มความเร็วในอัตราคงที่ $a_x = 5.0 \, \text{m/s}^2$ จากนั้นเคลื่อนที่ด้วยความเร็วคงที่ และสุดท้ายลดความเร็วลงในอัตราเดียวกับตอนแรกจนหยุดนิ่ง เวลาในการเคลื่อนที่ทั้งหมดเท่ากับ $25.0 \, \text{s}$ ความเร็วเฉลี่ยตลอดช่วงเวลานั้นเท่ากับ $72 \, \text{km/h}$ ถามว่ารถยนต์เคลื่อนที่ด้วยความเร็วคงที่เป็นเวลานานเท่าใด

เฉลย (วิธีแคลคูลัส):

ขั้นตอนที่ 1: แปลงหน่วยและหาข้อมูลพื้นฐาน

ความเร็วเฉลี่ย: $v_{avg} = 72 \, \frac{\text{km}}{\text{h}} \times \frac{1000 \, \text{m}}{1 \, \text{km}} \times \frac{1 \, \text{h}}{3600 \, \text{s}} = 20 \, \text{m/s}$

จากสมการความเร็วเฉลี่ย $\Delta x = v_{avg} \times t_{total}$ เราสามารถคำนวณระยะทางทั้งหมดได้:

ระยะทางทั้งหมด: $\Delta x_{total} = 20 \, \text{m/s} \times 25.0 \, \text{s} = 500 \, \text{m}$

จากความสมมาตรของการเคลื่อนที่ (อัตราเร่งเท่ากับอัตราหน่วง) ทำให้เวลาที่ใช้ในช่วงเร่งความเร็ว ($t_1$) และลดความเร็ว ($t_3$) จะเท่ากัน ดังนั้น $t_{total} = t_1 + t_2 + t_3 = 2t_1 + t_2 = 25.0 \, \text{s}$

ขั้นตอนที่ 2: สร้างฟังก์ชันความเร็วและหาระยะทางโดยการอินทิเกรต

ความเร็ว $v(t)$ หาได้จากความสัมพันธ์ $v(t) = \int a(t) \,dt$ และระยะทาง $\Delta x = \int v(t) \,dt$

ช่วงที่ 1 ($0 \le t \le t_1$): $v_1(t) = \int 5.0 \,dt = 5.0t$ และ $\Delta x_1 = \int_{0}^{t_1} 5.0t \,dt = 2.5t_1^2$

ช่วงที่ 2 ($t_1 \le t \le t_1+t_2$): ความเร็วคงที่ที่ $v_{max} = v_1(t_1) = 5.0t_1$ ดังนั้น $\Delta x_2 = v_{max} \cdot t_2 = (5.0t_1)t_2$

ช่วงที่ 3: เนื่องจากความสมมาตร $\Delta x_3 = \Delta x_1 = 2.5t_1^2$

ขั้นตอนที่ 3: แก้ระบบสมการ

ผลรวมของระยะทางคือ: $\Delta x_{total} = \Delta x_1 + \Delta x_2 + \Delta x_3$

$500 = 2.5t_1^2 + 5.0t_1t_2 + 2.5t_1^2 \Rightarrow 500 = 5.0t_1^2 + 5.0t_1t_2$

หารด้วย 5.0 ทั้งสมการ: $100 = t_1^2 + t_1t_2 \Rightarrow 100 = t_1(t_1 + t_2)$ (สมการ A)

จากความสัมพันธ์ของเวลา เรามี $t_2 = 25.0 - 2t_1$ (สมการ B)

แทน (B) ใน (A): $100 = t_1(t_1 + (25.0 - 2t_1)) \Rightarrow 100 = t_1(25.0 - t_1)$

จัดรูปเป็นสมการกำลังสอง: $t_1^2 - 25.0t_1 + 100 = 0$ ซึ่งแก้ได้ $t_1 = 5.0 \, \text{s}$ (อีกคำตอบหนึ่งคือ 20s ซึ่งทำให้ $t_2$ ติดลบ)

นำค่า $t_1$ ไปหา $t_2$: $t_2 = 25.0 - 2(5.0) = 15.0 \, \text{s}$

คำตอบ: รถยนต์เคลื่อนที่ด้วยความเร็วคงที่เป็นเวลานาน **15.0 วินาที**

โจทย์: มีจุดสามจุดอยู่บนมุมของสามเหลี่ยมด้านเท่าซึ่งมีด้านยาวเท่ากับ $a$ ทั้งหมดเริ่มเคลื่อนที่พร้อมกันด้วยความเร็วขนาดคงที่ $v$ โดยที่จุดที่หนึ่งมุ่งไปยังจุดที่สองเสมอ, จุดที่สองมุ่งไปยังจุดที่สาม, และจุดที่สามมุ่งไปยังจุดที่หนึ่ง ถามว่าอีกนานเท่าใดจุดทั้งสามจึงจะมาบรรจบกัน

เฉลย (วิธีแคลคูลัสเชิงความเร็วสัมพัทธ์):

ขั้นตอนที่ 1: หาอัตราการเข้าใกล้กัน

พิจารณาจุด 1 และ 2 ซึ่งมีระยะห่างกัน $r$ อัตราการเปลี่ยนแปลงของระยะห่าง ($\frac{dr}{dt}$) คือผลต่างขององค์ประกอบความเร็วตามแนวเส้นตรงที่เชื่อมระหว่างจุดทั้งสอง

องค์ประกอบความเร็วของจุดที่ 1 ที่พุ่งเข้าหาจุดที่ 2 คือ $v$

มุมภายในของสามเหลี่ยมด้านเท่าคือ $60^\circ$ ดังนั้น องค์ประกอบความเร็วของจุดที่ 2 ที่พุ่งหนีจากจุดที่ 1 ตามแนวเชื่อมคือ $v \cos(60^\circ) = v/2$

อัตราการเปลี่ยนแปลงสุทธิของระยะห่างคือ: $\frac{dr}{dt} = v_{\text{component,2}} - v_{\text{component,1}} = \frac{v}{2} - v = -\frac{v}{2}$

เครื่องหมายลบแสดงว่าระยะห่างลดลงด้วยอัตราเร็วคงที่ $v_{approach} = v/2$

ขั้นตอนที่ 2: คำนวณเวลา

เนื่องจากอัตราเร็วในการเข้าใกล้กันมีค่าคงที่ เราจึงสามารถใช้สมการพื้นฐาน $t = \frac{\text{distance}}{\text{speed}}$

$$t = \frac{\text{ระยะห่างเริ่มต้น}}{\text{อัตราเร็วในการเข้าใกล้}} = \frac{a}{v/2}$$

คำตอบ: จุดทั้งสามจะมาบรรจบกันในเวลา $\frac{2a}{v}$

โจทย์: เรือข้ามฟากแม่น้ำกว้าง $d$ ด้วยความเร็วเทียบกับน้ำ ($v_b$) ซึ่งน้อยกว่าความเร็วกระแสน้ำ ($v_w$) อยู่ $\eta=2$ เท่า ($v_w = 2v_b$) จะต้องเคลื่อนที่ทำมุมเท่าใดเทียบกับทิศทางของกระแสน้ำ เพื่อให้มีระยะเบี่ยง (drifting) มากที่สุด

เฉลย (วิธีแคลคูลัสและการแปลงแบบกาลิเลโอ):

ขั้นตอนที่ 1: นิยามกรอบอ้างอิงและเวกเตอร์ในสมการ $\vec{v}(t) = \vec{U} + \vec{v}'(t)$

เราจะใช้สมการการแปลงความเร็วแบบกาลิเลโอ โดยนิยามแต่ละส่วนดังนี้:

  • $\vec{v}(t)$ คือความเร็วของเรือเทียบกับฝั่ง (ความเร็วลัพธ์ที่เราสังเกตเห็น)
  • $\vec{U}$ คือความเร็วของกรอบอ้างอิงที่เคลื่อนที่ (กระแสน้ำ) เทียบกับฝั่ง: $\vec{U} = v_w \hat{i}$
  • $\vec{v}'(t)$ คือความเร็วของเรือเทียบกับกรอบอ้างอิงที่เคลื่อนที่ (น้ำ): $\vec{v}'(t) = (v_b \cos\theta) \hat{i} + (v_b \sin\theta) \hat{j}$

แทนค่าลงในสมการ: $\vec{v}(t) = (v_w \hat{i}) + \big( (v_b \cos\theta) \hat{i} + (v_b \sin\theta) \hat{j} \big)$

รวมองค์ประกอบจะได้: $\vec{v}(t) = (v_w + v_b \cos\theta) \hat{i} + (v_b \sin\theta) \hat{j}$

ขั้นตอนที่ 2: สร้างฟังก์ชันระยะเบี่ยง $x(\theta)$

เวลาที่ใช้ข้ามฟาก ($t$) ขึ้นอยู่กับระยะทางแนวตั้ง ($d$) และความเร็วแนวตั้ง ($v_y$) เท่านั้น:

$t = \frac{d}{v_y} = \frac{d}{v_b \sin\theta}$

ระยะเบี่ยง ($x$) คือระยะทางแนวนอนที่เคลื่อนที่ได้ในเวลา $t$:

$x = v_x \cdot t = (v_w + v_b \cos\theta) \left( \frac{d}{v_b \sin\theta} \right)$

จากโจทย์ $v_w = 2v_b$ ดังนั้น: $x(\theta) = (2v_b + v_b \cos\theta) \left( \frac{d}{v_b \sin\theta} \right) = d \left( \frac{2 + \cos\theta}{\sin\theta} \right)$

ขั้นตอนที่ 3: หาอนุพันธ์เพื่อหาค่าสูงสุด

ใช้กฎการหาอนุพันธ์ของผลหาร $\frac{d}{d\theta}\left(\frac{u}{v}\right) = \frac{u'v - uv'}{v^2}$:

$\frac{dx}{d\theta} = d \left[ \frac{(-\sin\theta)(\sin\theta) - (2+\cos\theta)(\cos\theta)}{\sin^2\theta} \right] = d \left[ \frac{-\sin^2\theta - 2\cos\theta - \cos^2\theta}{\sin^2\theta} \right]$

จากเอกลักษณ์ $\sin^2\theta + \cos^2\theta = 1$: $\frac{dx}{d\theta} = d \left[ \frac{-(\sin^2\theta + \cos^2\theta) - 2\cos\theta}{\sin^2\theta} \right] = d \left[ \frac{-1 - 2\cos\theta}{\sin^2\theta} \right]$

ให้ $\frac{dx}{d\theta} = 0$ แสดงว่าตัวเศษต้องเป็นศูนย์: $-1 - 2\cos\theta = 0 \Rightarrow \cos\theta = -\frac{1}{2}$

คำตอบ: มุมที่ทำให้ $\cos\theta = -1/2$ ในช่วง $0^\circ < \theta < 180^\circ$ คือ **$120^\circ$**

โจทย์: พิสูจน์ว่าค่า $g$ สามารถคำนวณได้จากสมการ $g = \frac{8h}{T_A^2 - T_B^2}$ เมื่อ $T_A, T_B$ คือช่วงเวลาที่วัตถุอยู่เหนือระดับ A และ B ตามลำดับ และ $h$ คือระยะห่างระหว่างระดับทั้งสอง

เฉลย (วิธีสมมาตรของการเคลื่อนที่):

ขั้นตอนที่ 1: วิเคราะห์เวลาจากจุดสูงสุด

เวลาที่วัตถุอยู่เหนือระดับ B คือ $T_B$ จากความสมมาตร เวลาที่ใช้ตกจากจุดสูงสุดมายังระดับ B คือ $t_B = T_B/2$

ในทำนองเดียวกัน เวลาที่ใช้ตกจากจุดสูงสุดมายังระดับ A คือ $t_A = T_A/2$

ขั้นตอนที่ 2: คำนวณระยะทางจากจุดสูงสุด

ใช้สมการการเคลื่อนที่แนวดิ่ง $s_y = u_y t + \frac{1}{2}gt^2$ โดยความเร็วต้น ณ จุดสูงสุด $u_y=0$

ระยะจากจุดสูงสุดถึง B: $h_B = \frac{1}{2}g(t_B)^2 = \frac{1}{2}g\left(\frac{T_B}{2}\right)^2 = \frac{gT_B^2}{8}$

ระยะจากจุดสูงสุดถึง A: $h_A = \frac{1}{2}g(t_A)^2 = \frac{1}{2}g\left(\frac{T_A}{2}\right)^2 = \frac{gT_A^2}{8}$

ขั้นตอนที่ 3: หาความสัมพันธ์และจัดรูปสมการ

ระยะห่างระหว่างสองระดับคือ $h = h_A - h_B$

$h = \frac{gT_A^2}{8} - \frac{gT_B^2}{8} = \frac{g}{8}(T_A^2 - T_B^2)$

จัดรูปสมการใหม่จะได้ $g = \frac{8h}{T_A^2 - T_B^2}$ ซึ่งเป็นสิ่งที่ต้องการพิสูจน์

โจทย์: ลูกบอลถูกปล่อยจากหยุดนิ่งจากความสูงแนวดิ่ง $h$ ลงบนพื้นเอียงเรียบที่ทำมุม $\alpha$ การกระดอนเป็นแบบยืดหยุ่นสมบูรณ์ จงหาระยะทาง $d$ ตามแนวพื้นเอียงที่ลูกบอลจะกระดอนไปกระทบพื้นเอียงเป็นครั้งที่สอง

เฉลย (วิธีโปรเจกไทล์ในแกนเอียง):

ขั้นตอนที่ 1: หาความเร็วก่อนกระทบ

จากกฎอนุรักษ์พลังงาน $mgh = \frac{1}{2}mv_1^2$ ความเร็วในแนวดิ่งก่อนกระทบคือ $v_1 = \sqrt{2gh}$

ขั้นตอนที่ 2: วิเคราะห์การชนในแกนเอียง

ตั้งแกน $x'$ ขนานพื้นเอียง, $y'$ ตั้งฉากพื้นเอียง การชนแบบยืดหยุ่นจะกลับทิศความเร็วในแนว $y'$ และคงความเร็วในแนว $x'$

ความเร็วหลังชน: $v_{2,x'} = v_1 \sin\alpha$ และ $v_{2,y'} = v_1 \cos\alpha$

ความเร่ง: $a_{x'} = g \sin\alpha$ และ $a_{y'} = -g \cos\alpha$

ขั้นตอนที่ 3: หาเวลาที่ลอยในอากาศ

ใช้สมการ $s_{y'} = u_{y'}t + \frac{1}{2}a_{y'}t^2$ โดยให้ $s_{y'}=0$: $0 = (v_1 \cos\alpha)t - \frac{1}{2}(g \cos\alpha)t^2$

แก้สมการได้ $t_{flight} = \frac{2v_1}{g}$

ขั้นตอนที่ 4: คำนวณระยะทาง $d$

ใช้สมการ $s_{x'} = u_{x'}t + \frac{1}{2}a_{x'}t^2$: $d = (v_1 \sin\alpha)t_{flight} + \frac{1}{2}(g \sin\alpha)t_{flight}^2$

$d = (v_1 \sin\alpha)\left(\frac{2v_1}{g}\right) + \frac{1}{2}(g \sin\alpha)\left(\frac{2v_1}{g}\right)^2 = \frac{4v_1^2 \sin\alpha}{g}$

คำตอบ: แทนค่า $v_1^2 = 2gh$ กลับเข้าไปจะได้ $d = \frac{4(2gh) \sin\alpha}{g} = 8h \sin\alpha$

โจทย์: บอลลูนลอยขึ้นจากผิวโลกด้วยอัตราเร็วแนวดิ่งคงที่ $v_0$ เนื่องจากลมทำให้มีความเร็วองค์ประกอบในแนวราบเป็น $v_x = ay$ โดย $a$ เป็นค่าคงที่ และ $y$ คือความสูง จงหา (ก) การเคลื่อนที่ $x(y)$ และ (ข) ความเร่งทั้งหมด, ความเร่งสัมผัส และความเร่งแนวตั้งฉาก

เฉลย (วิธีแคลคูลัส):

(ก) การเคลื่อนที่ $x(y)$:

จาก $dy/dt = v_0 \Rightarrow \int dy = \int v_0 dt \Rightarrow y(t) = v_0t$

จาก $dx/dt = ay = a(v_0t) \Rightarrow \int dx = \int av_0t \,dt \Rightarrow x(t) = \frac{1}{2}av_0t^2$

กำจัด $t$ โดยใช้ $t=y/v_0$: $x(y) = \frac{1}{2}av_0\left(\frac{y}{v_0}\right)^2 = \frac{a}{2v_0}y^2$

(ข) ความเร่ง:

$\vec{v}(t) = (av_0t)\hat{i} + v_0\hat{j} \Rightarrow \vec{a}(t) = \frac{d\vec{v}}{dt} = av_0\hat{i}$

$v = |\vec{v}| = \sqrt{(av_0t)^2 + v_0^2} = v_0\sqrt{a^2t^2+1}$

$a_t = \frac{dv}{dt} = \frac{v_0a^2t}{\sqrt{a^2t^2+1}}$ (ความเร่งแนวสัมผัส)

จาก $|\vec{a}|^2 = a_t^2 + a_n^2 \Rightarrow a_n = \sqrt{(av_0)^2 - a_t^2} = \frac{av_0}{\sqrt{a^2t^2+1}}$ (ความเร่งแนวตั้งฉาก)

คำตอบ: (ก) $x(y) = \frac{a}{2v_0}y^2$ และ (ข) $\vec{a} = av_0\hat{i}$, $a_t = \frac{v_0a^2t}{\sqrt{a^2t^2+1}}$, $a_n = \frac{av_0}{\sqrt{a^2t^2+1}}$

โจทย์: ปืนใหญ่ยิงกระสุนสองนัดด้วยความเร็วต้น $v_0 = 250 \, \text{m/s}$ นัดแรกทำมุม $\theta_1 = 60^\circ$ และนัดที่สองทำมุม $\theta_2 = 45^\circ$ จงหาช่วงเวลาระหว่างการยิง ($\Delta t$) ที่ทำให้กระสุนทั้งสองชนกัน

เฉลย (วิธีเคลื่อนที่สัมพัทธ์):

ขั้นตอนที่ 1: ตั้งเงื่อนไขการชน

การชนจะเกิดขึ้นได้ถ้าเวกเตอร์ตำแหน่งสัมพัทธ์ ($\vec{r}_{12}$) และเวกเตอร์ความเร็วสัมพัทธ์ ($\vec{v}_{12}$) อยู่บนแนวเส้นตรงเดียวกันและมีทิศตรงข้ามกัน

ขั้นตอนที่ 2: ตั้งสมการและแก้หา $\Delta t$

จากการวิเคราะห์การเคลื่อนที่สัมพัทธ์ ซึ่งมีความเร่งสัมพัทธ์เป็นศูนย์ ($\vec{a}_{12}=0$) จะสามารถพิสูจน์ได้ว่าช่วงเวลาที่ทำให้เกิดการชนคือ:

$$ \Delta t = \frac{2v_0 \sin(\theta_1 - \theta_2)}{g(\cos\theta_1 + \cos\theta_2)} $$

ขั้นตอนที่ 3: แทนค่า

$\sin(60^\circ-45^\circ) = \sin(15^\circ) \approx 0.2588$

$\cos(60^\circ) = 0.5$ และ $\cos(45^\circ) \approx 0.7071$

$$ \Delta t = \frac{2(250)(0.2588)}{9.8(0.5 + 0.7071)} = \frac{129.4}{11.829} \approx 10.94 \, \text{s} $$

คำตอบ: ช่วงเวลาระหว่างการยิงควรเป็นประมาณ **10.9 วินาที**

โจทย์: ลูกบอลกระทบกับพื้นเอียงที่ทำมุม $\theta$ ที่จุด 1 ด้วยความเร็ว $v_0$ ในแนวตั้งฉากกับพื้นเอียง แล้วกระดอนออกไปแบบโปรเจกไทล์จนกระทบพื้นเอียงที่จุด 2 (ระยะ $R_{12}$) จากนั้นกระดอนออกไปอีกครั้งจนกระทบที่จุด 3 (ระยะ $R_{23}$) กำหนดให้การชนเป็นแบบยืดหยุ่น จงหา (ก) ระยะ $R_{12}$ และ (ข) ระยะ $R_{23}$ และอัตราส่วน $R_{23}/R_{12}$

เฉลย (วิธีโปรเจกไทล์ในแกนเอียง):

(ก) หาระยะ $R_{12}$

ตั้งแกน $x'$ ขนานพื้นเอียง, $y'$ ตั้งฉาก ความเร็วเริ่มต้น $\vec{u}_1 = 0\hat{i'} + v_0 \hat{j'}$ ความเร่ง $\vec{a} = (g \sin\theta)\hat{i'} - (g \cos\theta)\hat{j'}$

หาเวลาที่ลอยในอากาศ ($t_1$) จาก $s_{y'} = u_{y'}t + \frac{1}{2}a_{y'}t^2$ โดยให้ $s_{y'}=0$:

$0 = v_0 t_1 - \frac{1}{2}(g\cos\theta)t_1^2 \Rightarrow t_1 = \frac{2v_0}{g\cos\theta}$

หาระยะ $R_{12}$ จาก $s_{x'} = u_{x'}t + \frac{1}{2}a_{x'}t^2$:

$R_{12} = (0)t_1 + \frac{1}{2}(g\sin\theta)t_1^2 = \frac{1}{2}g\sin\theta\left(\frac{2v_0}{g\cos\theta}\right)^2 = \frac{2v_0^2 \tan\theta}{g\cos\theta}$

(ข) หาระยะ $R_{23}$ และอัตราส่วน

หาความเร็วก่อนกระทบจุด 2 จาก $v = u+at$: $v_{x',1} = a_{x'}t_1 = 2v_0 \tan\theta$, $v_{y',1} = u_{y'} + a_{y'}t_1 = -v_0$

การชนยืดหยุ่นทำให้ความเร็วหลังชน (ความเร็วต้นของช่วงที่สอง) คือ $\vec{u}_2 = (2v_0 \tan\theta)\hat{i'} + v_0 \hat{j'}$

เวลาที่ลอย ($t_2$) จะเท่าเดิมเพราะ $u_{y'}$ และ $a_{y'}$ ไม่เปลี่ยน: $t_2 = t_1$

หาระยะ $R_{23}$ จาก $s_{x'} = u_{x'}t + \frac{1}{2}a_{x'}t^2$:

$R_{23} = u_{x',2}t_2 + \frac{1}{2}a_{x'}t_2^2 = (2v_0 \tan\theta)t_1 + R_{12} = 2(g\sin\theta t_1)\frac{t_1}{g\sin\theta} + R_{12} = 2R_{12} + R_{12} = 3R_{12}$

คำตอบ: (ก) $R_{12} = \frac{2v_0^2 \tan\theta}{g\cos\theta}$

(ข) $R_{23} = 3R_{12} = \frac{6v_0^2 \tan\theta}{g\cos\theta}$ และอัตราส่วน $R_{23}/R_{12} = 3$

โจทย์: แท่งไม้เริ่มเคลื่อนที่ขึ้นพื้นเอียงทำมุม $\beta$ ด้วยความเร็ว $v$ ในขณะเดียวกันลูกบอลถูกโยนขึ้นจากจุดเริ่มต้นเดียวกันด้วยความเร็ว $u$ ทำมุม $\theta$ กับแนวราบ จงหาสมการของมุม $\theta$ ที่ขึ้นอยู่กับ $\beta$ ที่ทำให้ลูกบอลตกบนแท่งไม้พอดี เมื่อแท่งไม้เคลื่อนที่ไปได้สูงสุดบนพื้นเอียง และหาค่า $\theta$ เมื่อ $\beta = 45^\circ$

เฉลย (วิธีซิงโครไนซ์การเคลื่อนที่):

ขั้นตอนที่ 1: วิเคราะห์การเคลื่อนที่ของแท่งไม้

ความเร่งของแท่งไม้คือ $a_p = -g\sin\beta$ จาก $v_f = v_i+at$, เวลาถึงจุดสูงสุด (ที่ $v_f=0$) คือ $t_{p} = \frac{v}{g\sin\beta}$

จาก $v_f^2 = v_i^2+2as$, ระยะทางสูงสุดคือ $s_{p} = \frac{v^2}{2g\sin\beta}$

พิกัดของจุดสูงสุดในแนวราบคือ $(x_p, y_p) = (s_{p}\cos\beta, s_{p}\sin\beta)$

ขั้นตอนที่ 2: ตั้งเงื่อนไขการชน

ลูกบอลต้องถึงพิกัด $(x_p, y_p)$ ในเวลา $t_p$ พอดี

$x_p = (u\cos\theta)t_{p}$ และ $y_p = (u\sin\theta)t_{p} - \frac{1}{2}gt_{p}^2$

ขั้นตอนที่ 3: แก้สมการหาความสัมพันธ์

จากสมการแกน x: $u\cos\theta = \frac{x_p}{t_{p}} = \frac{s_{p}\cos\beta}{t_{p}} = \frac{(v^2/2g\sin\beta)\cos\beta}{v/g\sin\beta} = \frac{v}{2}\cos\beta$

จากสมการแกน y: $u\sin\theta = \frac{y_p + \frac{1}{2}gt_{p}^2}{t_{p}} = \frac{s_p\sin\beta + \frac{1}{2}g t_{p}^2}{t_{p}} = \frac{(v^2/2g\sin\beta)\sin\beta + \frac{1}{2}g(v/g\sin\beta)^2}{v/g\sin\beta} = \frac{v}{2\sin\beta}(\sin^2\beta+1)$

หา $\tan\theta = \frac{u\sin\theta}{u\cos\theta} = \frac{(\sin^2\beta+1)/\sin\beta}{\cos\beta} = \frac{\sin^2\beta+1}{\sin\beta\cos\beta}$

ขั้นตอนที่ 4: หาค่า $\theta$ เมื่อ $\beta=45^\circ$

$\tan\theta = \frac{\sin^2(45^\circ)+1}{\sin(45^\circ)\cos(45^\circ)} = \frac{(1/\sqrt{2})^2+1}{(1/\sqrt{2})(1/\sqrt{2})} = \frac{1/2+1}{1/2} = 3$

คำตอบ: สมการความสัมพันธ์คือ $\tan\theta = \frac{\sin^2\beta+1}{\sin\beta\cos\beta}$ และเมื่อ $\beta=45^\circ$ จะได้ $\theta = \arctan(3) \approx 71.57^\circ$

โจทย์: ถังเปล่าถูกดึงให้ลากปลายโซ่ที่วางบนพื้น โดยระบบเคลื่อนที่ด้วยความเร่งคงที่ $a$ โซ่มีความหนาแน่นเชิงเส้น $\lambda$ (kg/m) จงหา (ก) แรงดึง $F(t)$ (ข) งานที่ทำโดยแรง $F(t)$ ในเวลา $t$ และ (ค) กำลังที่สูญเสียไป ณ เวลา $t$

เฉลย (วิธีระบบมวลแปรค่า):

(ก) หาแรง $F(t)$

ใช้กฎข้อที่สองของนิวตันรูปทั่วไป $F_{ext} = \frac{dp}{dt}$ โดยที่ $p = mv$

ณ เวลา $t$: $v(t) = at$, $x(t) = \frac{1}{2}at^2$, $m(t) = \lambda x(t) = \frac{1}{2}\lambda at^2$

โมเมนตัมของระบบ: $p(t) = m(t)v(t) = (\frac{1}{2}\lambda at^2)(at) = \frac{1}{2}\lambda a^2 t^3$

แรง: $F(t) = \frac{dp}{dt} = \frac{d}{dt}(\frac{1}{2}\lambda a^2 t^3) = \frac{3}{2}\lambda a^2 t^2$

(ข) หางานที่ทำโดยแรง $F(t)$

งานหาจาก $W = \int P dt$ โดยกำลังคือ $P(t) = F(t)v(t)$

$P(t) = (\frac{3}{2}\lambda a^2 t^2)(at) = \frac{3}{2}\lambda a^3 t^3$

$W(t) = \int_0^t \frac{3}{2}\lambda a^3 (t')^3 dt' = [\frac{3}{8}\lambda a^3 (t')^4]_0^t = \frac{3}{8}\lambda a^3 t^4$

(ค) หากำลังที่สูญเสียไป

กำลังที่ใส่เข้าระบบ ($P_{in}=P(t)$) จะเปลี่ยนไปเป็นอัตราการเพิ่มของพลังงานจลน์ ($\frac{dK}{dt}$) และกำลังที่สูญเสียไป ($P_{lost}$)

พลังงานจลน์: $K(t) = \frac{1}{2}m(t)v(t)^2 = \frac{1}{2}(\frac{1}{2}\lambda at^2)(at)^2 = \frac{1}{4}\lambda a^3 t^4$

อัตราการเพิ่มของพลังงานจลน์: $\frac{dK}{dt} = \frac{d}{dt}(\frac{1}{4}\lambda a^3 t^4) = \lambda a^3 t^3$

$P_{lost} = P_{in} - \frac{dK}{dt} = \frac{3}{2}\lambda a^3 t^3 - \lambda a^3 t^3 = \frac{1}{2}\lambda a^3 t^3$

คำตอบ: (ก) $F(t) = \frac{3}{2}\lambda a^2 t^2$

(ข) $W(t) = \frac{3}{8}\lambda a^3 t^4$

(ค) $P_{lost}(t) = \frac{1}{2}\lambda a^3 t^3$
